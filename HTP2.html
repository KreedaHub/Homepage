2048

Objective

Slide numbered tiles on a 4x4 grid to combine them and reach the 2048 tile.

Game Rules

  • The game is played on a 4x4 grid.
  • Use arrow keys (or swipe on mobile) to move tiles in one direction.
  • When two tiles with the same number touch, they merge into one.
  • The goal is to create a tile with the number 2048.
  • The game ends when there are no more moves left.

How to Play

Swipe or use arrow keys to slide tiles. Tiles move in the selected direction until they hit the edge or another tile. If two tiles with the same number touch, they merge into a new tile with double the value.

Winning Conditions

Reach the **2048 tile** to win. You can continue playing to get even higher numbers.
